# Docker Commands - clinic-database

## Build
docker build -t ozanemrahyakupoglu/clinic-database:1.0.1 .

## Push (Docker Hub)
docker push ozanemrahyakupoglu/clinic-database:1.0.1

## Run (migrate)
docker run --rm \
  -e FLYWAY_URL="jdbc:postgresql://46.225.97.71:5432/postgres?currentSchema=clinic" \
  -e FLYWAY_USER=postgres \
  -e FLYWAY_PASSWORD=XXX \
  ozanemrahyakupoglu/clinic-database:1.0.1
